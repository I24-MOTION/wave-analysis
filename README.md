# wave-analysis
A tool for analyzing stop-and-go waves in I-24 MOTION INCEPTION v1.0 data.

## Repository Structure

- `data/`: Input data.
- `demo/`: Example scripts demonstrating tool usage.
- `results/`: Output results.
